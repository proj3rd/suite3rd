# suite3rd

Suite for telecommunications development

## ASN.1

### ASN.1 extractor

Extract ASN.1 definition from plain text or Word document

#### CLI

```sh
npx suite3rd asn1 extract <filepath>
```

## 3GPP standardization

### Spec downloader

Download 3GPP spec

#### CLI

```sh
npx suite3rd 3gpp fetch spec <specNumber> <release> <date>
# e.g. npx suite3rd 3gpp fetch spec 38.331 17 2023-09
```
